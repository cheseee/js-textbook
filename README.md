# 📘 Учебник по JavaScript

Персональный учебник, который я пишу вместе с ИИ-агентом. Не готовая книга — а **живой конспект**, который растёт вместе со мной.

## Как пользоваться этим учебником

1. **Читаю главу** слева направо. Не перескакиваю.
2. **Запускаю примеры кода** сам — копирую в браузер (DevTools → Console) или в Node.
3. **Делаю упражнения** из папки `exercises/` — без подглядывания в ответ.
4. **Отмечаю непонятное** в `my-notes.md` и прошу агента объяснить.
5. **Возвращаюсь** — перечитываю, когда забыл.

> Главное правило: **если не понимаю строку — останавливаюсь и спрашиваю.** Не иду дальше, пока не стало ясно.
